O programa foi desenvolvido usando a linguagem de progração Python, com a ferramenta Qt Designer para criação da interface e Sqlite como o sistema de gerenciamento do banco de dados. O sistema consiste em um controle de Estoque, ele tem as  funções de adicionar, excluir e alterar a quantidades guardadas no banco de dados.
